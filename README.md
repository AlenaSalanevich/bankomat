# Bankomat
Добрый день!)
Несколько дополнений... или скорее придуманных мною ограничений по заданию: 
1. Входные данные: db-файл с кратким описанием карт-счета: пин-код карты, сумма на карт-счету.
(для примера создано четыре карт-счета:
1111, 1000

2. Из статистики исключаем только те местоименияя, союзы и предлоги, которые перечисленны в отдельном txt-файле stopwords.txt. Все что туда не попало, из статистики не исключается. Stopwords.txt  идут в "комплекте" с приложением :)
3.Глаголы в разных participle считаются разными словами, существительные во множественном и единственном числе тоже разные слова.
4. Цифры (даты, количество) принимаем за слова.
